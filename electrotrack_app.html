// CÓMO INSTALAR EN MÓVIL (GRATIS)

Esta app funciona como PWA (Progressive Web App) y se puede instalar gratis en iOS y Android para hasta 3 personas.

📱 EN IPHONE / IPAD (Safari)
  1. Abre esta página en Safari
  2. Toca el ícono de Compartir (cuadro con flecha)
  3. Selecciona "Agregar a pantalla de inicio"
  4. Ponle nombre y toca Agregar ✓
🤖 EN ANDROID (Chrome)
  1. Abre esta página en Chrome
  2. Toca los 3 puntos arriba a la derecha
  3. Selecciona "Agregar a pantalla de inicio" o "Instalar app"
  4. Confirma y listo ✓

⚠️ App diseñada para 3 usuarios: 1 jefe + 2 trabajadores. Los datos se guardan localmente en cada dispositivo.
